# PicacgSanctuary
用于自动下载哔咔庇护所本子和里番的脚本

## 使用者 > 
使用此脚本需要安装火狐浏览器
然后在Release下载可执行文件

## 开发者 > 
本脚本使用到selenium等模块，没安装的要使用pip安装哟。因为开发的时候没用虚拟环境emmmm要是缺包了就自己找把。
可以在release下载到原项目包和可执行包。

```
pip install selenium
pip install yaml
pip install mitmproxy
pip install m3u8
pip install cryptography
```
